# 快速上手

本指南将帮助您在5分钟内开始使用账清应用。

## 第一次启动

### 1. 创建家庭组

首次启动应用时，您需要创建一个家庭组：

1. 点击"创建家庭组"按钮
2. 输入家庭组名称（例如：我的家庭）
3. 点击"确定"

### 2. 添加账户

创建您的第一个账户：

1. 进入"账户"页面
2. 点击右下角的"+"按钮
3. 填写账户信息：
   - 账户名称（例如：支付宝）
   - 账户类型（支付宝/微信/银行卡/现金等）
   - 初始余额
4. 点击"保存"

---

## 记录交易

### 手动添加交易

1. 在首页点击"+"按钮
2. 选择交易类型（支出/收入）
3. 填写交易信息：
   - 金额
   - 分类（点击选择）
   - 账户
   - 日期和时间
   - 备注（可选）
4. 点击"保存"

### 导入账单

支持从CSV/Excel文件或邮箱导入账单：

1. 进入"导入"页面
2. 选择导入方式：
   - **文件导入**: 选择CSV或Excel文件
   - **邮箱导入**: 配置IMAP邮箱自动获取账单
3. 映射字段（首次导入需要）
4. 预览并确认导入

---

## 分类管理

### 使用系统分类

应用预置了常用分类：
- 支出：餐饮、交通、购物、娱乐等
- 收入：工资、奖金、投资等

### 自定义分类

1. 进入"分类"页面
2. 点击"+"添加新分类
3. 设置分类信息：
   - 名称
   - 类型（支出/收入）
   - 图标和颜色
   - 父分类（可选）

### 智能分类规则

设置关键词规则自动分类交易：

1. 进入"分类"页面
2. 选择一个分类，点击"规则"
3. 添加关键词（例如：美团 → 餐饮）
4. 设置优先级和置信度

---

## AI智能分类

### 配置AI模型

1. 进入"设置" → "AI设置"
2. 点击"添加模型配置"
3. 选择AI提供商（通义千问/DeepSeek）
4. 输入API密钥（会自动加密存储）
5. 自定义提示词（可选）

### 使用AI分类

1. 在交易列表中选择未分类的交易
2. 点击"AI分类"按钮
3. 等待AI分析并返回建议分类
4. 确认或修改分类

---

## 数据分析

### 查看统计

1. 进入"分析"页面
2. 查看各种统计图表：
   - 收支趋势
   - 分类占比
   - 账户分布
3. 切换时间范围（本月/本年/自定义）

### 预算管理

1. 进入"预算"页面
2. 设置年度预算
3. 查看月度预算执行情况
4. 接收预算预警提醒

---

## 数据备份

### 本地备份

1. 进入"设置" → "备份与同步"
2. 点击"导出备份"
3. 选择保存位置
4. 备份文件包含所有数据（加密）

### WebDAV同步

配置WebDAV实现多设备同步：

1. 进入"设置" → "备份与同步"
2. 启用"WebDAV同步"
3. 填写服务器信息：
   - 服务器地址
   - 用户名和密码
4. 点击"测试连接"
5. 设置自动同步间隔
