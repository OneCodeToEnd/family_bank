# 备份与同步

账清支持本地备份和WebDAV同步，保护您的数据安全。

## 本地备份

### 导出备份

1. 进入"设置" → "备份与同步"
2. 点击"导出备份"
3. 选择保存位置
4. 备份文件自动加密

### 备份内容

备份文件包含：
- 所有交易记录
- 账户信息
- 分类设置
- 规则配置
- 应用设置

### 导入备份

1. 进入"设置" → "备份与同步"
2. 点击"导入备份"
3. 选择备份文件
4. 输入密码（如已加密）
5. 确认导入

!!! warning "注意"
    导入备份会覆盖当前数据，请谨慎操作！

---

## WebDAV同步

### 什么是WebDAV

WebDAV是一种网络文件传输协议，支持：
- 坚果云
- Nextcloud
- ownCloud
- 自建WebDAV服务器

### 配置WebDAV

1. 进入"设置" → "备份与同步"
2. 启用"WebDAV同步"
3. 填写服务器信息：
   - **服务器地址**：完整URL（如 https://dav.jianguoyun.com/dav/）
   - **用户名**：WebDAV账户
   - **密码**：WebDAV密码或应用密码
   - **同步路径**：可选，指定同步目录
4. 点击"测试连接"
5. 保存配置

### 常见WebDAV服务

**坚果云：**
- 服务器：https://dav.jianguoyun.com/dav/
- 需要在网页端生成应用密码

**Nextcloud：**
- 服务器：https://your-domain.com/remote.php/dav/files/username/
- 使用Nextcloud账户密码

## 同步策略

### 自动同步

配置自动同步：
- 启用"自动同步"
- 设置同步间隔（15分钟/30分钟/1小时/手动）
- 应用会在后台自动同步

### 手动同步

随时手动触发同步：
1. 进入"备份与同步"页面
2. 点击"立即同步"
3. 等待同步完成

### 冲突处理

当多设备同时修改数据时：
- 以最新修改为准
- 保留冲突版本
- 提示用户手动合并

## 数据加密

### 备份加密

本地备份支持加密：
- 使用AES-256加密
- 设置备份密码
- 密码不可找回，请妥善保管

### 传输加密

WebDAV同步：
- 使用HTTPS加密传输
- 服务器端可选加密存储
- 保护数据隐私安全

## 最佳实践

### 定期备份

- 每周至少备份一次
- 重要操作前先备份
- 保留多个备份版本

### 多设备同步

- 使用WebDAV实现多设备同步
- 避免同时在多设备操作
- 定期检查同步状态

### 安全建议

- 使用强密码
- 启用备份加密
- 定期更换WebDAV密码
- 不要在公共网络同步

## 故障排除

### 备份导出失败

**问题**：无法导出备份文件

**解决方案**：
- 检查存储权限
- 确保有足够空间
- 尝试更换保存位置

### WebDAV连接失败

**问题**：无法连接WebDAV服务器

**解决方案**：
- 验证服务器地址格式
- 检查用户名和密码
- 确认网络连接正常
- 查看服务器日志

### 同步冲突

**问题**：多设备数据不一致

**解决方案**：
- 手动合并冲突数据
- 选择保留哪个版本
- 避免同时操作
