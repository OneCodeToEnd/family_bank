# 交易记录

交易记录是账清的核心功能，记录您的每一笔收支。

## 创建交易

### 手动添加

1. 在首页点击"+"按钮
2. 选择交易类型：
   - **支出**：记录消费支出
   - **收入**：记录收入来源
   - **转账**：账户间转账
3. 填写交易信息：
   - **金额**：交易金额（必填）
   - **分类**：选择交易分类（必填）
   - **账户**：选择支付账户（必填）
   - **日期时间**：交易发生时间
   - **交易对方**：商家或收款方（可选）
   - **备注**：补充说明（可选）
4. 点击"保存"

### 快速记账

使用快捷方式快速记账：
- 首页长按"+"按钮
- 选择常用分类
- 输入金额即可保存

## 查看交易

### 交易列表

在交易列表页可以：
- 查看所有交易记录
- 按时间倒序排列
- 显示金额、分类、账户等信息

### 筛选和搜索

**按时间筛选：**
- 今天
- 本周
- 本月
- 本年
- 自定义时间范围

**按类型筛选：**
- 全部
- 支出
- 收入
- 转账

**按账户筛选：**
- 选择特定账户
- 查看该账户的交易

**按分类筛选：**
- 选择特定分类
- 查看该分类的交易

**搜索：**
- 输入关键词
- 搜索备注、交易对方等字段

---

## 编辑交易

### 修改交易

1. 点击交易记录
2. 进入交易详情页
3. 点击"编辑"按钮
4. 修改交易信息
5. 保存更改

### 删除交易

1. 在交易详情页点击"删除"
2. 确认删除操作
3. 账户余额自动更新

### 批量操作

选择多笔交易进行批量操作：
- **批量删除**：删除多笔交易
- **批量分类**：统一修改分类
- **批量导出**：导出为CSV文件

---

## 交易去重

### 自动去重

系统使用SHA-256哈希自动去重：
- 基于：账户 + 时间 + 金额 + 描述
- 导入时自动检测重复
- 避免重复记账

### 手动处理

如果发现重复交易：
1. 在交易列表中找到重复项
2. 对比交易详情
3. 删除重复的交易

---

## 交易确认

### 未确认交易

导入的交易默认为"未确认"状态：
- 显示特殊标记
- 需要人工审核
- 确认后才计入统计

### 确认交易

1. 查看交易详情
2. 核对信息是否正确
3. 点击"确认"按钮
4. 交易状态变为"已确认"

### 批量确认

1. 选择多笔未确认交易
2. 点击"批量确认"
3. 快速完成审核

---

## 交易统计

### 基础统计

在交易列表顶部显示：
- **总收入**：选定时间范围内的总收入
- **总支出**：选定时间范围内的总支出
- **净收入**：收入减去支出
- **交易笔数**：交易记录数量

### 详细分析

进入"分析"页面查看：
- 收支趋势图
- 分类占比
- 账户分布
- 月度对比

---

## 交易导入

### 支持格式

- **CSV文件**：逗号分隔值文件
- **Excel文件**：.xlsx 或 .xls 格式
- **邮箱账单**：从邮箱自动获取

详见 [账单导入](bill-import.md)

---

## 交易导出

### 导出数据

1. 进入交易列表
2. 点击"导出"按钮
3. 选择导出格式（CSV/Excel）
4. 选择时间范围
5. 保存文件

### 导出内容

导出文件包含：
- 交易时间
- 交易类型
- 金额
- 分类
- 账户
- 交易对方
- 备注
- 确认状态
