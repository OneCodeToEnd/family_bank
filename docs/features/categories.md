# 分类系统

账清使用层级分类树结构，帮助您精确分类每一笔交易。

## 分类结构

### 层级设计

分类采用两级结构：
- **一级分类**：大类（如"餐饮"、"交通"）
- **二级分类**：细分类（如"早餐"、"午餐"、"晚餐"）

### 分类类型

- **支出分类**：记录各类支出
- **收入分类**：记录各类收入

---

## 系统预设分类

### 支出分类

- **餐饮**：早餐、午餐、晚餐、零食、饮料
- **交通**：公交、地铁、打车、加油、停车
- **购物**：服饰、日用品、电子产品、图书
- **娱乐**：电影、游戏、旅游、运动
- **居住**：房租、物业、水电、网络
- **医疗**：挂号、药品、体检
- **教育**：学费、培训、书籍
- **社交**：礼物、聚餐、红包
- **其他**：未分类支出

### 收入分类

- **工资**：月薪、奖金、补贴
- **投资**：股票、基金、理财收益
- **兼职**：副业收入
- **礼金**：红包、礼物
- **其他**：未分类收入

## 自定义分类

### 创建分类

1. 进入"分类"页面
2. 点击"+"添加分类
3. 填写分类信息：
   - **名称**：分类名称
   - **类型**：支出或收入
   - **父分类**：选择上级分类（可选）
   - **图标**：从图标库选择
   - **颜色**：自定义分类颜色
   - **标签**：添加关键词标签
4. 保存分类

### 编辑分类

1. 点击要编辑的分类
2. 修改分类信息
3. 保存更改

### 删除分类

!!! warning "注意"
    - 系统预设分类不可删除
    - 删除分类不会删除相关交易，交易会变为"未分类"
    - 删除父分类会同时删除所有子分类

---

## 分类规则

### 关键词匹配

设置关键词规则实现自动分类：

**示例规则：**
- "美团" → 餐饮
- "滴滴" → 交通
- "京东" → 购物
- "工资" → 工资收入

### 添加规则

1. 选择一个分类
2. 点击"规则"按钮
3. 添加关键词
4. 设置优先级和置信度
5. 保存规则

### 规则优先级

- **高优先级**：优先匹配
- **中优先级**：常规匹配
- **低优先级**：兜底匹配

### 置信度

- **高置信度（≥80%）**：自动分类
- **中置信度（50-80%）**：建议分类，需确认
- **低置信度（<50%）**：仅作参考

## 智能分类

### 自动学习

系统会从您的分类行为中学习：
1. 记录您的手动分类
2. 分析分类模式
3. 自动优化匹配规则
4. 提高分类准确度

### 批量分类

对未分类的交易进行批量分类：
1. 选择多笔未分类交易
2. 点击"批量分类"
3. 系统自动匹配分类
4. 确认或调整分类结果

---

## 分类管理

### 隐藏分类

不常用的分类可以隐藏：
- 隐藏后不在选择列表中显示
- 已有交易不受影响
- 可随时恢复显示

### 排序

自定义分类显示顺序：
1. 进入分类管理
2. 长按拖动分类
3. 调整顺序
4. 保存更改

---

## 分类统计

### 查看统计

在分类详情页可查看：
- **交易笔数**：该分类的交易数量
- **总金额**：累计金额
- **平均金额**：单笔平均金额
- **占比**：在同类型中的占比
- **趋势图**：按时间的金额趋势

### 预算设置

为分类设置预算：
1. 进入分类详情
2. 点击"设置预算"
3. 输入月度预算金额
4. 系统会在超支时提醒

---

## 最佳实践

### 分类原则

- **简洁明了**：避免过度细分
- **符合习惯**：按个人消费习惯分类
- **定期整理**：删除不用的分类

### 规则设置

- **精准关键词**：使用特征明显的关键词
- **避免冲突**：不同分类的关键词不要重复
- **定期优化**：根据使用情况调整规则
