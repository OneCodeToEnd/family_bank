# Git 提交历史

查看最近的 Git 提交记录，以清晰的格式展示。

## 执行内容

运行 `git log` 命令，显示：
- 提交哈希（短格式）
- 提交日期
- 提交作者
- 提交信息

如果用户提供了数字参数，显示指定数量的提交记录，否则默认显示 10 条。

格式示例：
```
1fb52fd (2026-01-24) feat: 实现邮件账单导入功能
853133e (2026-01-23) 增加数据分析功能
c6bcebd (2026-01-22) llm category
```
